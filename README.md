
## restaurant-reservation

A simple Python CLI restaurant reservation system using SQLite.

## Requirements
- Python 3.10+
- openpyxl

## Install
py -m pip install -r requirements.txt

## Run
py restaurant.py

## Features
- Table management
- Reservations with time conflict check
- Excel / CSV export
- Persistent database
